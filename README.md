# cli-breathing

- Basic usage (3 minutes, square breathing, circle visualization)

- Custom duration (in seconds), pattern, and style

- Available patterns:
  - square (4-4-4-4)
  - relaxed (4-7-8-0)
  - zen (6-0-6-0)
  - energy (2-0-2-0)

- Available styles:
  - circle
  - wave
  - lotus
  - zen

# Instructions

To install dependencies: 

```bash
bun install
```


To build: 

```
bun run build
```

To run: 
```
./dist/breathe
```
